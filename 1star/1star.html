Instructions

For this task we will explore how to create a centered camera

Create the enviroment

To start create an enviroment to walk around in. We can do this in a few ways but for this task I would recomend doing the following:

  1. Find a background image to form a room or a floor
  2. Use preload() to load in the image
  3. In draw() use image() to draw the background. For now use a hardcoded x and y position.

Create a player

Next we need a player. For this task you may want to use a Class but for understanding how the offset works I suggest doing the following:

  1. In draw() or in a custom playerDraw() function create a rect().
  2. Create 2 global variables for the X and Y values. Set their values in setup().
  3. Set the width and height to 50, 50.

Add the offset

Now that our level is set up we can add the required offsets to make our level move

  1. For the background image edit the image() function X and Y values to follow this formular: Starting Pos + Canvas Offset - Player Pos for example 50 + width/2 - x
  2. For the player edit the rect() function X and Y values to follow this formular: Starting Pos + Canvas Offset - Player Pos for example x + width/2 - x
  3. Try out the game and see the momvent.
