Instructions

For this task we will learn how to create a 'lookahead' camera. For this task some code to create the level has been provided. If you run the code you will see we have a centred camera

Create the variables

To create our lookahead camera system we need to add 4 global variables:

  1. Create lookaheadX and lookaheadY at the top of the script. Set their value to 0.

Setting the Value of lookaheadX & lookaheadY

lookaheadX and lookaheadY set our direction telling us which way we need the player to move so that there is more room ahead of them.

  1. In playerMovement() add the lookaheadX value to each if() statement. The value for each should follow the direction of travel. For example, the first if() statement will have a lookaheadX value of -1 and lookaheadY of 0. This is the A key which will move the player left.

Add the offset

Now we have our variables we can set the lookahead offset.

  1. In the drawPlayer() function create two local variables called offsetX and offsetY.
  2. Set the value of the two new local variables will is lookaheadX * OFFSET AMOUNT and lookaheadY * OFFSET AMOUNT. In this task, OFFSET AMOUNT is a value of your choice.
  3. For the player edit the rect() function X and Y values to follow this formular: Starting Pos + Canvas Offset - Player Pos - Look Ahead Camera Offset for example x + width/2 - x - offsetX
  4. Try out the game and see the movement.

Add Lerp

You will see that the player now jumps backwards when we walk in different directions. This is the effect we are looking for but it does not look very smooth!

To improve on this do the following:

  1. Create cameraX and cameraY at the top of the script. Set their value to 0.
  2. In the drawPlayer() function set the value of cameraX and cameraY to be a lerp() which takes in cameraX as the starting position, offsetX as the end positions and 0.1 as the iteration. For example, for X the code will be cameraX = lerp(cameraX, offsetX, 0.1);
  3. Edit the player rect() to replace offsetX and offsetY with the new cameraX and cameraY values.
  4. Try out the game and see smoother movement
